# Upload to GitHub - Step by Step

## Option 1: Using GitHub Web Interface (Easiest)

1. **Go to GitHub.com** and sign in as "morninrage"
2. **Click the "+" icon** (top right) → **"New repository"**
3. **Repository name**: `beyond-bell-game`
4. **Description**: "Beyond Bell Space Shooter Game with Leaderboard"
5. **Make it Public** (or Private if you prefer)
6. **DO NOT** check "Initialize with README"
7. **Click "Create repository"**

### Upload Files:

1. On the new repository page, you'll see "uploading an existing file"
2. **Click "uploading an existing file"**
3. **Drag and drop** the entire `bell-game` folder contents:
   - config.js
   - game.js
   - index.html
   - README.md
   - server/ (entire folder)
   - All other files
4. **Scroll down**, add commit message: "Initial commit - Beyond Bell game"
5. **Click "Commit changes"**

Done! Your code is on GitHub!

## Option 2: Install GitHub Desktop (Recommended for future)

1. Download: https://desktop.github.com/
2. Install and sign in
3. File → Add Local Repository → Select `bell-game` folder
4. Publish to GitHub

## Option 3: Install Git Command Line

1. Download: https://git-scm.com/download/win
2. Install (use default options)
3. Then we can use git commands
